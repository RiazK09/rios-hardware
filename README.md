# Rio's Hardware

## About the Project

Rio's Hardware is a static webpage that displays a selection of hardware tools.

I created the page using HTML, CSS and Bootstrap. The products are displayed in 
a responsive grid, with each product showing an image, name, price and Buy button.

This website was created for an assignment and is not a real online store.

## Features

- Responsive product grid
- Product images, names and prices
- Buy buttons for each product
- Hover effects on the product cards
- Header with the website name
- Footer with a contact form
- Mobile-friendly layout

## Technologies Used

- HTML
- CSS
- Bootstrap 5

## Project Files

- `index.html` contains the webpage content.
- `style.css` contains the custom styling.
- The `images` folder contains the product images.

## How to Open the Website

1. Download the project folder.
2. Open the folder.
3. Double-click `index.html` to open the website in a browser.

An internet connection is needed because Bootstrap is loaded using a CDN.

## Deployment

The website is hosted using GitHub Pages.

Live website: https://riazk09.github.io/rios-hardware/

## Important Note

The Buy buttons and contact form are only included as part of the webpage design. 
They do not process orders or send information.
